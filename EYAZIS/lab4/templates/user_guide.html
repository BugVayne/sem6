User Manual for Sentence Viewer Application

Introduction

The Sentence Viewer application allows users to upload RTF files containing sentences, view stored sentences, and add new sentences directly through the web interface.

Features

Home Page

  • Welcome Message: Displays a welcoming message and provides navigation to view sentences.
  • See Sentences Button: Click this button to navigate to the sentences page.

Uploading Files

  1. Upload Section:
    • Click the "Choose File" button to select an RTF file from your device.
    • After selecting a file, the button will change to "Upload File".
    • Click the "Upload File" button to submit the file.
  2. File Requirements:
    • Only RTF files are accepted.
    • If the file format is invalid, an error message will be displayed.
  3. Feedback:
    • Success or error messages will be displayed based on the upload result.

Adding Sentences

  1. Input Section:
    • Enter your sentence in the provided text box.
    • Click the "Add Sentence" button to submit the sentence.
  2. Feedback:
    • Confirmation messages will inform you whether the sentence was added successfully.

Viewing Sentences

After uploading files or adding sentences, click the "See Sentences" button to view all saved sentences.

Error Handling

The application provides feedback for common issues, such as:

  • No file selected during upload.
  • Invalid file format.
  • Empty sentence when adding a new one.
