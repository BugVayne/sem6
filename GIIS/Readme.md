#Отчет о лабораторных работах по предмету ГИИС

В рамках лабораторных работ было необходимо разработать приложение для взаимодействия с различными графическими обьектами и алгоритмами.
Для реализации пользовательского интерфейса приложения была выбрана библиотека PyQt5.


Для взаимодействия с определенным набором функций для определенной лабораторной работы следует выбрать пункт меню с соответствущей темой.

### Реализованные алгоритмы  
- **Алгоритм Брезенхема** (целочисленный, эффективный)  
- **ЦДА (Цифровой Дифференциальный Анализатор)** (интерполяция с плавающей точкой)  
- **Алгоритм Ву** (сглаживание ступенчатости)  

### Интерактивность  
- Клик левой кнопкой - установка **начальной точки**  
- Второй клик - установка **конечной точки** → автоматическое построение линии  
- Привязка к сетке для точного позиционирования  

### Настройки визуализации  
- **Режим отладки**:  
  - Пошаговая отрисовка пикселей  
  - Вывод координат в консоль  
  - Замедленная визуализация (задержка 50 мс на шаг)  
- **Автоочистка холста** после построения (можно отключить)  
- Настраиваемый **размер пикселя** (по умолчанию 10px)
